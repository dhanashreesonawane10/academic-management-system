About the Project

Smart academic management with a modern tech experience

Project Overview

The College Management System (CAMS) is a web-based platform designed to simplify communication and academic task handling between students and faculty. It provides a unified portal for managing assignments, notices, and academic updates in a smooth, organized, and digital way.

Objective

The main objective of this project is to build a simple, reliable, and user-friendly system that reduces paperwork, saves time, and improves academic workflow using modern web technologies.

Key Features

  • Secure login system for students and faculty
  • Faculty can upload assignments and notices
  • Students can view and submit assignments
  • Digital notice board for announcements
  • Separate dashboards for faculty and students
  • Lightweight and fast SQLite database

Technologies Used

Layer Technology
Frontend HTML, CSS, JavaScript
Backend Python (Flask Framework)
Database SQLite
Tools VS Code, DB Browser for SQLite

Future Scope

  • Marks & grading module
  • Email notification system
  • Attendance tracking system
