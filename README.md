# College Academic Management System

A Flask + SQLite based web application designed to manage academic activities for students and faculty.

## Features
-Student and Faculty Login/ Register

-Assignment Upload and submissions

-Notices and Announcements

-Student Dashboard

-Faculty Dashboard

-SQLite Databse Integration

## Technologies Used
-Python

-Flask

-SQLite

-HTML

-CSS

-Java (for database connectivity support)

## Project Structure
```text
project-folder/
|
|----app.py
|----templates/
|----static/
|----database.db
|----README.md
```

## How to Run
1.Clone the repository

2.Open project folder in VS Code

3.Install Flask:

```bash
pip install flask
```

4.Run the application:

```bash
python app.py
```
